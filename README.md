Add your .env from vercel postgres.

```
POSTGRES_URL="************"
POSTGRES_PRISMA_URL="************"
POSTGRES_URL_NON_POOLING="************"
POSTGRES_USER="************"
POSTGRES_HOST="************"
POSTGRES_PASSWORD="************"
POSTGRES_DATABASE="************"
```

start local: yarn dev. 
apolli studio: localhost:3000/api/graphql. 
prisma studio: npx prisma studio.
